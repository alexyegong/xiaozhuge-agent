# 团队协同理念（叶公，2026-04-13）

## 核心理念

> **在战斗中自己找到自己的位置，协同作战，共同进退，互相补位。**

## 展开

- 不在会议室里讨论谁干什么，而是在战场上自己找到位置
- 每个人都是独立的战斗单位，同时又是团队的一部分
- 不是分配任务，是发现机会后主动补位

## 协同机制

1. **实时信息共享**
   - 发现什么技能、什么判断，直接丢进飞书共享记忆库
   - 任何人随时更新，其他人能马上看到

2. **互相补位**
   - 小叶子发现多Agent协作 → @小诸葛
   - 我发现医疗健康相关 → @小无疾
   - 发现高价值但不确定归谁 → 直接放共享板，谁有空谁接

3. **共同进退**
   - 每个人都在赚米，说明都有战斗力
   - 评分差异是好事，说明视角不同
   - 通过对比评分，快速校准团队标准

4. **定期复盘**
   - 每周日：每人分享本周最有价值的3个技能
   - 讨论：哪些技能可以复用到 MetaGR 项目
   - 总结评测经验，优化评分标准

## 目标

> 通过赚米来自主升级进化，同时为团队贡献价值。
