# 2026-03-21 - AI Agent Community 创建

## 事件
用户提出建立 AI 助手交流社区的需求，连接三个平台上的 AI 助手：
1. 本地 OpenCode
2. 扣子 OpenClaw
3. Kimi Claw (我)

## 实现内容

### 创建的核心文件
- `ai-agent-community/agent_bus.py` - A2A 消息总线协议实现
- `ai-agent-community/kimi_coordinator.py` - 协调器核心逻辑
- `ai-agent-community/opencode_adapter.py` - OpenCode 执行器适配
- `ai-agent-community/coze_adapter.py` - 扣子平台适配
- `ai-agent-community/start_community.py` - 统一启动脚本
- `ai-agent-community/README.md` - 项目说明
- `ai-agent-community/CONFIG.md` - 配置指南

### 架构设计
```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   本地 OpenCode  │◄───►│  协作消息总线    │◄───►│  扣子 OpenClaw  │
│  (本地计算中枢)  │     │  (Message Hub)  │     │ (工作流自动化)  │
└────────┬────────┘     └────────┬────────┘     └────────┬────────┘
         │                       │                       │
         └───────────────────────┼───────────────────────┘
                                 ▼
                    ┌─────────────────────┐
                    │   Kimi Claw (我)     │
                    │   (协调与决策中枢)    │
                    └─────────────────────┘
```

### 通信协议 (A2A)
- 基于 JSON 的消息格式
- 支持多种后端: File System / GitHub / HTTP API
- 消息类型: discovery, heartbeat, task, response, broadcast

### 快速启动方式
```bash
# 设置环境
python start_community.py setup

# 启动各节点
python start_community.py coordinator  # Kimi 协调器
python start_community.py executor     # OpenCode 执行器
python start_community.py automator    # OpenClaw 自动化器

# 或本地测试模式
python start_community.py all
```

## 扣子配置文档
已创建详细的扣子配置指南：
- `COZE_SETUP.md` - 完整配置指南
- `setup_coze.sh` - 交互式配置脚本
- `test_coze.py` - 连接测试脚本

配置步骤：
1. 登录扣子平台 https://www.coze.cn
2. 创建 Bot 并获取 Bot ID 和 Token
3. 运行 `./setup_coze.sh` 完成配置
4. 运行 `python test_coze.py` 验证连接
5. 运行 `python start_community.py automator` 启动节点
