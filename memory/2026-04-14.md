# 2026-04-14 项目启动

## 辽宁瑞慈医疗集团融资项目

### 项目背景
叶公发起，为辽宁鞍山医疗机构融资项目。瑞慈旗下四大板块：
1. 辽宁瑞慈残疾人托养院
2. 辽宁瑞慈临终关怀医院
3. 辽宁瑞慈残疾儿童康复院
4. 辽宁瑞慈残疾人就业培训学校

### 我方定位
- 技术提供方：无疾科技（MetaGR Hunter体系）
- 核心能力：142万人次健康监测数据 + 数字中医康复方案
