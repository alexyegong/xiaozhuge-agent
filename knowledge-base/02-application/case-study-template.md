# 临床案例库模板

## 案例格式标准

每个案例应包含以下字段：

```yaml
案例ID: CASE-YYYY-NNNN
建档日期: YYYY-MM-DD
最后更新: YYYY-MM-DD
案例类型: [疼痛/睡眠/心血管/肿瘤/精神/其他]

# 基本信息
患者信息:
  性别: [男/女]
  年龄: XX岁
  职业: 
  既往病史: 
  家族病史: 
  生活方式: [饮食/运动/睡眠/压力]

# 检测数据
检测记录:
  - 日期: YYYY-MM-DD
    设备: MetaGR Hunter
    操作员: 
    主要发现:
      - 部位: 
        风险等级: [97%/85%/71%/57%/42%/28%]
        可能疾病: 
        熵值: 
    电磁场特征:
      基线频率: 
      异常频段: 
      相位状态: 

# 干预方案
干预记录:
  - 日期: YYYY-MM-DD
    干预类型: [ELF脉冲/生活方式/药物辅助]
    参数设置:
      频率组合: 
      强度: 
      时长: 
    配合措施:
      - 

# 效果评估
随访记录:
  - 日期: YYYY-MM-DD（干预后X天）
    主观反馈:
      症状变化: 
      生活质量评分: 1-10
    客观指标:
      复查结果: 
      熵值变化: 
      风险等级变化: 
    不良事件: [无/有，说明]

# 案例总结
效果评级: [显著/有效/轻微/无效]
起效时间: X天/周/月
完全康复: [是/否/进行中]
关键成功因素: 
经验教训: 

# 案例标签
标签:
  - 

# 隐私声明
患者授权: [已签署知情同意书]
数据脱敏: [是]
可分享级别: [内部/学术/公开]
```

## 案例示范

### 案例一：急性疼痛即时缓解

```yaml
案例ID: CASE-2026-0001
建档日期: 2026-03-01
案例类型: 疼痛

患者信息:
  性别: 男
  年龄: 45岁
  职业: 办公室职员
  既往病史: 久坐腰痛3年
  生活方式: 久坐，运动少

检测记录:
  - 日期: 2026-03-01
    主要发现:
      - 部位: 腰椎L4-L5
        风险等级: 85%
        可能疾病: 椎间骨关节病
        熵值: 高

干预记录:
  - 日期: 2026-03-01
    干预类型: ELF脉冲
    参数设置:
      频率: 7.83Hz（舒曼共振）
      时长: 30分钟

随访记录:
  - 日期: 2026-03-01（即时）
    主观反馈:
      症状变化: 疼痛从8分降至3分
      生活质量评分: 7
    效果评级: 显著

案例总结:
  效果评级: 显著
  起效时间: 即时
  关键成功因素: 精准定位+舒曼共振频率
```

## 数据收集标准

### 必须收集
- [ ] 患者基本信息（性别、年龄）
- [ ] 检测原始数据（频率、熵值、风险等级）
- [ ] 干预参数（频率、时长、部位）
- [ ] 主观反馈（症状评分、生活质量）
- [ ] 不良事件记录

### 建议收集
- [ ] 生活方式问卷
- [ ] 血液指标（如有）
- [ ] 影像学检查（如有）
- [ ] 心理状态评估

### 可选收集
- [ ] 基因信息
- [ ] 肠道菌群
- [ ] 环境因素

## 案例使用规范

### 学术研究
- 需伦理委员会批准
- 数据脱敏处理
- 统计分析需注明局限性

### 教学培训
- 可用于L3-L5级培训
- 需说明典型性和代表性

### 商业推广
- 需患者明确授权
- 不得夸大效果
- 必须平衡展示（包括无效案例）
