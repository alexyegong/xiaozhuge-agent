# 生命电磁场理论（LEFT）框架

## 理论定位

**LEFT** = Life Electromagnetic Field Theory

一个试图整合以下学科的跨学科理论框架：
- 现代生物物理学
- 信息论与系统科学
- 中医整体观
- 电磁场工程

---

## 核心假设

### 假设1：生命的电磁本质

生命体可以被视为一个复杂的电磁系统：
- 细胞膜维持电位差（生物电池）
- 神经系统传递电信号
- 心脏、大脑产生强电磁场
- 细胞间通过电磁场通讯

### 假设2：健康即有序

从信息论视角：
- **健康** = 高有序性（低熵状态）
- **疾病** = 有序性下降（熵增）
- **康复** = 恢复有序性（减熵）

### 假设3：频率即信息

电磁场的特征携带健康信息：
- 频率分布 → 系统状态
- 相位关系 → 协调性
- 振幅变化 → 活跃程度

### 假设4：共振即沟通

生命系统与环境的电磁交互：
- 地球舒曼共振 → 生命节律基准
- 外源ELF脉冲 → 调节干预手段
- 同频共振 → 信息传递机制

## 核心方程

### 1. 健康状态函数

```
H = f(O, I, S)

其中：
H = Health (健康状态)
O = Orderliness (有序性)
I = Information flow (信息流通度)
S = Self-healing activation (自愈机制激活度)
```

### 2. 疾病风险函数

```
R = g(F_a, P_d, E)

其中：
R = Risk (疾病风险)
F_a = Frequency anomaly (频率异常)
P_d = Phase disorder (相位紊乱)
E = Entropy increase (熵增)
```

### 3. 干预效果函数

```
E = h(F_m, T, I_d)

其中：
E = Effect (干预效果)
F_m = Frequency matching (频率匹配度)
T = Timing (时机选择)
I_d = Individual difference (个体差异)
```
