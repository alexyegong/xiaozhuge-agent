# 五行与系统动力学

## 中医五行理论

### 基本概念
> 五行：木、火、土、金、水，代表五种基本物质和运动形态。

### 五行属性

| 五行 | 特性 | 方位 | 季节 | 脏腑 |
|-----|------|------|------|------|
| **木** | 生发、条达 | 东 | 春 | 肝、胆 |
| **火** | 炎上、温热 | 南 | 夏 | 心、小肠 |
| **土** | 承载、生化 | 中 | 长夏 | 脾、胃 |
| **金** | 收敛、肃杀 | 西 | 秋 | 肺、大肠 |
| **水** | 寒凉、润下 | 北 | 冬 | 肾、膀胱 |

### 五行关系

**相生**（促进）：
```
木 → 火 → 土 → 金 → 水 → 木
```

**相克**（制约）：
```
木 → 土 → 水 → 火 → 金 → 木
```

## 电磁场视角的五行

### 频率-五行对应

| 五行 | 频率范围 | 电磁场特征 |
|-----|---------|-----------|
| **木**（生发） | 3-4Hz | 上升沿、增长趋势 |
| **火**（炎上） | 8-9Hz | 峰值、高频活跃 |
| **土**（中和） | 5-7Hz | 平稳段、基线稳定 |
| **金**（收敛） | 2-3Hz | 下降沿、频率收敛 |
| **水**（潜藏） | 1-2Hz | 低幅、低频稳定 |

*注：具体数值为初步假设，需实验验证*

### 相生相克的物理意义

**相生 = 能量/信息的正向传递**：
```
木(3-4Hz) → 火(8-9Hz)：频率跃升，能量激发
火(8-9Hz) → 土(5-7Hz)：能量转化，趋于稳定
土(5-7Hz) → 金(2-3Hz)：进一步收敛，有序化
金(2-3Hz) → 水(1-2Hz)：能量潜藏，储备状态
水(1-2Hz) → 木(3-4Hz)：重新激发，开始新一轮
```

**相克 = 负反馈调节**：
```
木克土：生发过旺抑制过度稳定
土克水：稳定平台防止过度潜藏
水克火：潜藏状态抑制过度活跃
火克金：活跃状态打破过度收敛
金克木：收敛控制防止过度生发
```

## 脏腑系统的五行网络

### 电磁场耦合模型

**假设**：
脏腑之间存在电磁耦合，形成复杂的调控网络。

**网络拓扑**：
```
        心（火）
       ↗    ↘
肝（木）←——→ 肺（金）
   ↓  ↘    ↗  ↓
   胆    脾（土）   大肠
    ↘   ↗
      肾（水）
```

**耦合强度**：
- 相生关系：正向耦合（+）
- 相克关系：负向耦合（-）

### 脏腑疾病的电磁传导

**示例：肝火上炎**（木火过旺）
```
病机：肝（木）→ 心（火）过度传导
电磁表现：
- 肝区：高频活跃（木→火）
- 心区：频率过高（火亢盛）
- 整体：阴阳失调，熵增

干预策略：
1. 滋水涵木（肾→肝）：降低肝区频率
2. 清心泻火：直接降低心区频率
3. 培土生金（脾→肺）：增强制约力量
```

## 临床应用

### 诊断：五行辨证

**步骤**：
1. 检测各脏腑电磁场频率
2. 绘制"五行频率图谱"
3. 识别异常五行（过旺或过衰）
4. 分析相生相克关系

**示例诊断**：
```
检测发现：
- 肝区：频率过高（木旺）
- 脾区：频率过低（土虚）
- 关系：木克土过度

诊断：肝郁脾虚
治则：疏肝健脾
```

### 治疗：五行调频

**基本原则**：
1. **补母泻子**：虚则补其母，实则泻其子
2. **抑强扶弱**：抑制过旺，扶助不足
3. **整体调和**：注重五行整体平衡

**频率干预策略**：
| 情况 | 干预方法 |
|-----|---------|
| 某行过旺 | 使用相克频率抑制 |
| 某行不足 | 使用相生频率补益 |
| 整体失调 | 使用中频（土）调和 |
